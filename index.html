Hoova Mobile

Version 1.0.0 is now available

Hoova is a simple quick window search app, you can:

  • Search directly from several search engines while doing other things
  • Translate from one language to the other in a click of a button
  • Use Speech-to-Text
  • Open page in web browser
  • Copy text
  • Save text and other cool functionality
